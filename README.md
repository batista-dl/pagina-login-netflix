Uma recriação da **página de login da Netflix**, desenvolvida com **HTML e CSS**, focada em design, organização de código.
Este projeto foi criado com o objetivo de praticar **front-end**, explorando:
- Estrutura semântica em HTML  
- Estilização moderna com CSS  
- Layout inspirado na interface original da Netflix  

Objetivo:
Aprimorar habilidades em desenvolvimento web, especialmente na construção de interfaces visuais responsivas e bem organizadas.

# Aprendizados
- Posicionamento de elementos  
- Uso de flexbox  
- Boas práticas de organização de código  

Desenvolvido para fins de estudos.
